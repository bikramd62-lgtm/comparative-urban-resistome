# Data availability

This repository contains code, metadata, processed matrices, derived analysis tables, and final figures associated with the master's thesis:

**Comparative Metagenomic Profiling of Urban Resistomes in Transit Surfaces and Wastewater Systems**

Raw sequencing reads are not included in this repository because of file size, licensing, and privacy-related considerations for metagenomic sequencing data.

The repository does not include raw FASTQ files, human-depleted read files, SAM/BAM alignment files, Bowtie2 index files, downloaded database files, or large intermediate outputs.

The repository includes selected derived files required to inspect and reproduce the downstream analysis, including city-level metadata, processed presence/absence matrices, summary tables, statistical outputs, and figure-generating scripts.

## Data sources

- Sewage metagenomic data: Global Sewage Surveillance project datasets
- Public-transit metagenomic data: MetaSUB/GeoSeeq-derived city-level data
- ARG database: MEGARes v3.00

## Included data types

- Matched city metadata
- Accession-to-city mapping files
- City-level covariate tables
- Processed ARG presence/absence matrices
- RQ1, RQ2, and RQ3 summary tables
- Final statistical outputs
