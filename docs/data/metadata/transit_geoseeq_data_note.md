# Transit GeoSeeq data note

Public-transit metagenomic data used in this thesis were obtained directly from the GeoSeeq platform rather than through ENA/SRA-style run accession downloads.

For this reason, the transit dataset is not represented here using run accession IDs in the same format as the sewage dataset. Transit data were selected at the city level according to the matched-city design used in the thesis.

The final matched transit cities correspond to the same 16-city panel with 5 runs for each city used for the sewage-transit comparison. Raw transit sequencing reads are not included in this repository because of file size, platform access, and data-sharing considerations.

This repository provides downstream metadata, processed matrices, analysis scripts, and derived outputs required to inspect the thesis analysis.
